# @core
## @kaiusdiego


Este é o core do sbf-converter, uma implementação que cria abstrações para entidades, repositórios, casos de uso, entre outras, que servem de base para o restante da aplicação.

## Abordagens utilizadas

- Doman-Driven Design
- Clean Architecture
- SOLID
