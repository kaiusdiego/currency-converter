# @core
## @kaiusdiego


This is the core of currency-converter, an implementation that creates abstractions for entities, repositories, use cases, among others, that serve as the basis for the rest of the application.

## Approaches used

- Doman-Driven Design
- Clean Architecture
- SOLID
